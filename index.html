leaflyene house 운영플래너 사용 가이드

각 탭을 클릭해 기능별 사용법을 확인하세요.

🚀

3단계로 바로 시작하기

URL에 접속하면 모든 팀원이 동일한 데이터를 실시간으로 공유합니다. 별도 설치나 로그인 없이 바로 사용할 수 있습니다.

1
분류(카테고리)를 먼저 만드세요

왼쪽 업무 추가 폼 상단의 ⚙ 분류 관리 버튼을 클릭해 메뉴, 마케팅, 인사 등 업무에 맞는 분류를 추가합니다. 기본 분류 7개가 이미 만들어져 있으니 그대로 사용해도 됩니다.

2
업무를 추가하세요

왼쪽 ✚ 업무 추가 패널에서 업무 제목을 입력하고, 분류와 기간을 선택한 뒤 ✚ 추가하기를 누릅니다. 추가 즉시 전체 팀원 화면에 반영됩니다.

3
캘린더·타임라인으로 일정을 확인하세요

가운데 패널 상단의 캘린더 / 타임라인 버튼으로 뷰를 전환합니다. 캘린더는 날짜별, 타임라인은 월간 Gantt 차트로 업무를 확인할 수 있습니다.

화면 구성 한눈에 보기
왼쪽 패널업무 추가 폼 — 새 업무를 입력하는 곳
가운데 패널캘린더 / 타임라인 뷰 — 일정 전체를 시각적으로 확인
오른쪽 패널주간 업무 — 이번 주 7일간의 업무 목록
하단 전체업무 목록 — 분류 필터, 긴급 알림, 상하위 계층 그룹 표시
상단 헤더전체 / 완료 / 초과 / 진행률 통계 + 팀 바로가기 링크 + 링크 모음
캘린더·타임라인뷰 위 분류 필터 탭으로 특정 분류 일정만 필터링해서 확인 가능
🔥 실시간 공유: Firebase 기반으로 누가 업무를 추가·수정·삭제하면 같은 URL을 열고 있는 모든 팀원 화면에 1초 이내에 자동 반영됩니다.
✚

업무 추가하기

화면 왼쪽 패널에서 새 업무를 입력합니다. 제목만 필수이고 나머지는 모두 선택 사항입니다.

업무 제목 *필수어떤 업무인지 간단하게 입력합니다. 예: "메뉴판 디자인 수정"
분류색상 칩을 클릭해 분류를 선택합니다. 여러 개 동시에 선택 가능합니다.
기간 (시작 ~ 마감)날짜를 입력하면 캘린더와 타임라인에 자동 표시됩니다. 하루짜리 업무는 시작일만 입력해도 됩니다.
상위 업무기존 업무의 하위 항목으로 만들고 싶을 때 선택합니다. 선택하지 않으면 독립 업무로 등록됩니다.
참고 링크관련 URL을 붙여넣으면 업무 카드와 링크 모음에서 바로 열 수 있습니다.
메모업무에 대한 추가 설명이나 체크리스트를 자유롭게 입력합니다.
💡 빠른 추가: 캘린더에서 날짜를 클릭하면 그 날짜가 자동으로 입력된 업무 추가 폼이 팝업으로 열립니다. 특정 날짜 업무를 빠르게 등록할 때 유용합니다.
🔗

하위 업무 (계층 구조)

큰 업무를 여러 단계 세부 항목으로 나눠 관리합니다. 타임라인에서 상위-하위 관계가 들여쓰기로 시각화됩니다.

방법 1
타임라인에서 바로 추가

타임라인 왼쪽 업무명 위에 마우스를 올리면 나타나는 + 버튼을 클릭합니다. 상위 업무 정보와 분류가 자동으로 채워진 하위 업무 추가 팝업이 열립니다.

방법 2
업무 추가 폼에서 연결

왼쪽 업무 추가 폼의 상위 업무 선택란에서 기존 업무를 선택한 뒤 추가하면 하위 업무로 등록됩니다.

방법 3
수정 팝업에서 변경

기존 업무의 수정 팝업에서 상위 업무를 선택하거나 제거해 계층을 언제든 바꿀 수 있습니다.

타임라인 표시하위 업무는 상위 업무 바로 아래에 들여쓰기(└)와 연한 배경으로 표시됩니다.
정렬 우선순위하위 업무가 있는 상위 업무가 목록 상단에 먼저 표시됩니다.
💡 팁: 하위 업무는 1단계까지만 지원합니다. 프로젝트 → 세부 작업 단위로 구성하는 것을 추천합니다.
🏷

분류(카테고리) 관리

업무를 색상과 아이콘으로 구분하는 분류를 직접 만들고 편집할 수 있습니다.

1
분류 관리 열기

왼쪽 패널의 ⚙ 분류 관리 버튼을 클릭합니다.

2
새 분류 추가

아이콘(이모지)과 분류 이름을 입력하고, 원하는 색상을 선택한 뒤 ✚ 분류 추가를 누릅니다.

3
기존 분류 수정 / 삭제

목록에서 분류 이름 옆 ✏️ 버튼으로 이름을 수정하거나, 🗑 버튼으로 삭제합니다. 삭제된 분류는 기존 업무에서 자동으로 제거됩니다.

⚠️ 주의: 분류 변경은 모든 팀원에게 즉시 반영됩니다. 분류를 삭제하면 해당 분류가 설정된 모든 업무에서 분류 정보가 제거됩니다.
📅

월간 캘린더 보기

한 달 전체 일정을 날짜 격자로 확인합니다. 날짜 클릭으로 해당 날 업무를 빠르게 조회·추가할 수 있습니다.

뷰 전환가운데 패널 상단 캘린더 탭을 클릭합니다.
분류 필터캘린더 위 분류 탭을 클릭하면 해당 분류의 업무만 캘린더에 표시됩니다. 전체를 누르면 전체 업무로 돌아옵니다. 타임라인 뷰도 동일한 필터가 적용됩니다.
월 이동제목 좌우 ‹ › 버튼으로 이전/다음 달로 이동합니다.
날짜 클릭클릭하면 팝업이 열려 그 날의 업무 목록 확인 + 새 업무를 바로 추가할 수 있습니다.
색상 점(dot)날짜 셀 아래 작은 점이 해당 날 업무의 분류 색상을 나타냅니다.
오늘 표시오늘 날짜 셀은 커피색 테두리로 강조됩니다.
💡 팁: 날짜 팝업 오른쪽 폼에서 날짜가 자동 입력된 채로 업무를 추가할 수 있어 일정 기반 등록에 가장 빠릅니다.
📊

타임라인 (Gantt) 보기

업무 기간을 막대로 시각화해 월간 일정 흐름을 한눈에 파악합니다.

뷰 전환가운데 패널 상단 타임라인 탭을 클릭합니다.
분류 필터캘린더/타임라인 위 분류 탭에서 특정 분류만 골라 보면 해당 분류 업무 막대만 표시됩니다.
완료 숨김우측 상단 완료 숨김 버튼을 누르면 체크 완료된 업무가 타임라인에서 사라집니다. 다시 누르면 전체보기 중으로 바뀌며 완료 업무도 표시됩니다.
월 이동제목 좌우 ‹ › 버튼으로 이전/다음 달로 이동합니다.
막대 클릭Gantt 막대를 클릭하면 해당 업무의 수정 팝업이 열립니다.
가로 스크롤Gantt 막대 영역만 가로로 스크롤되고, 왼쪽 업무명 목록은 고정됩니다.
정렬 순서하위 업무가 있는 상위 업무 → 단독 업무 순으로, 시작일 오름차순 정렬됩니다.
날짜 미정기간이 없는 업무는 목록 하단 "날짜 미정" 구역에 모아서 표시됩니다.
하위 업무 추가왼쪽 업무명에 마우스를 올리면 나타나는 + 버튼으로 하위 업무를 바로 만들 수 있습니다.
💡 팁: 분류 필터 + 완료 숨김을 함께 쓰면 특정 카테고리의 진행 중인 업무만 골라서 깔끔하게 확인할 수 있습니다.
🗓

주간 업무 확인

오른쪽 패널에서 이번 주 월~일 7일간 진행 중인 업무를 날짜별로 확인합니다.

주 이동좌우 ‹ › 버튼으로 이전/다음 주로 이동합니다.
이번 주로↩ 이번 주로 버튼을 누르면 오늘 기준 주로 바로 이동합니다.
표시 기준해당 주의 날짜 범위 안에 포함되는 업무가 요일별로 정렬되어 표시됩니다.
완료 업무완료 처리된 업무는 취소선과 함께 흐리게 표시됩니다.
💡 팁: 주간 패널은 1100px 이하 화면에서는 자동으로 숨겨집니다. 넓은 화면에서 가장 유용합니다.
📋

업무 목록 · 수정 · 삭제

화면 하단에서 모든 업무를 필터링하고, 카드를 클릭해 수정·삭제합니다. 상하위 관계가 계층 구조로 시각화됩니다.

🚨 기간 초과 알림목록 상단에 마감일이 지난 업무가 빨간색 카드로 강조 표시됩니다. D+n일 초과 정보가 함께 보입니다.
🔥 마감 임박 알림마감 3일 이내 업무는 주황색 카드로 표시되며, D-n 정보가 함께 보입니다. 두 섹션 모두 클릭해 접을 수 있습니다.
상하위 계층 구조상위 업무 카드 아래에 들여쓰기된 세로선과 함께 하위 업무 카드가 바로 붙어 표시됩니다. 어떤 업무가 어디에 속하는지 한눈에 파악할 수 있습니다.
진행률 바전체 업무 중 완료된 비율을 퍼센트와 막대로 표시합니다.
분류 필터상단 탭 버튼으로 특정 분류만 필터링합니다. 전체 탭으로 전체 목록을 봅니다.
완료 토글카드 왼쪽 체크 아이콘으로 완료/미완료를 전환합니다. 완료된 업무는 흐리게 표시됩니다.
수정 팝업카드 또는 타임라인 막대를 클릭하면 제목·분류·기간·링크·메모를 모두 수정할 수 있는 팝업이 열립니다.
삭제수정 팝업 하단 삭제 버튼을 누릅니다. 삭제는 복구되지 않으며, 모든 팀원에게 즉시 반영됩니다.
링크 버튼참고 링크가 있는 업무는 카드에 링크 버튼이 표시되어 바로 열 수 있습니다.
⚠️ 삭제 주의: 상위 업무를 삭제해도 하위 업무는 삭제되지 않습니다. 하위 업무 연결이 끊어진 채 독립 업무로 남습니다.
🔗

링크 모음

참고 링크가 등록된 업무를 한 곳에서 모아 봅니다. 관련 자료를 빠르게 찾을 때 유용합니다.

1
상단 바의 🔗 링크 모음 버튼 클릭

링크가 등록된 업무 전체를 분류별로 정리한 모달이 열립니다.

2
분류별 링크 확인

링크는 업무 분류(카테고리)별로 그룹화되어 표시됩니다. 각 항목에는 업무명과 링크 도메인이 보입니다.

3
열기 ↗ 버튼으로 바로 이동

항목 오른쪽 열기 버튼을 클릭하면 해당 링크가 새 탭에서 열립니다.

링크 추가 방법업무 추가 또는 수정 팝업의 참고 링크 항목에 URL을 입력하면 자동으로 링크 모음에 포함됩니다.
완료된 업무완료 처리된 업무의 링크는 업무명에 취소선이 표시됩니다.
상단 바로가기상단 헤더의 📝 회의록 / 📁 구글드라이브 / 💬 슬랙 탭으로 팀 공용 문서에 빠르게 접근할 수 있습니다.
💡 팁: 업무마다 링크를 넣어두면 "그 자료 어디 있더라?" 를 찾아 헤맬 필요 없이 링크 모음에서 바로 찾을 수 있습니다.
🔥

실시간 동기화 & 구글 캘린더 연동

Firebase로 팀 전체 실시간 공유, 구글 캘린더로 개인 일정까지 관리합니다.

🔥 Firebase 실시간 동기화
작동 방식URL 접속 즉시 Firebase에서 최신 데이터를 자동으로 불러옵니다. 별도 로그인이 필요 없습니다.
실시간 반영누군가 업무를 추가·수정·삭제하면 같은 URL을 열고 있는 모든 화면에 1초 이내로 자동 반영됩니다.
동기화 배지상단 헤더의 🔥 실시간 동기화 배지가 켜져 있으면 정상 동기화 중입니다.
📅 구글 캘린더 연동
최초 연동상단 🔗 구글 캘린더 연동 버튼 클릭 → 구글 계정으로 로그인 → 🌿 leaflyene 운영플래너 전용 캘린더가 자동 생성되고 모든 업무가 동기화됩니다.
자동 재연결한 번 연동하면 다음 방문 시 버튼을 누르지 않아도 자동으로 재연결됩니다.
